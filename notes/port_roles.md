# Port Roles
## Root Port (RP)
  - To reach The Root bridge
## Designated Port (DP)
  - Forwarding port
  - One per Link
## Blocking/Non-Designated Port (Loops)
  - loop

## why
We needed the labeling to identify loops and we have space for other links so use defined other roles
