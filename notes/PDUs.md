# PDU

## what
PDU = Protocol data units

The data unit names as referred to data -> segments -> packets -> frame


- data
  - at the upper layers
- segments
  - at the Transport Layer
- packets
  - at the Network layers
- frame
  - at the datalink
- bit 
  - at the physical layer
